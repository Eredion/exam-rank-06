mini_serv

funciones permitidas: close socket bind select write recv atoi strcpy memset
					strlen send exit sprintf bzero strstr malloc calloc free

Programa que funcione como servidor para que se conecten diferentes clientes.
Debe escuchar solo en un puerto (argumento).

Cada cliente que se conecte tiene un id que es un int (empieza por 0 y va sumando.)

Cuando llegue un mensaje lo tiene que reenviar a todos los demas clientes con un formato
específico que indica el cliente que lo ha enviado.

Cuando un cliente se va hay que notificarlo. Mirar también para escribir en el error sterr
si falla algo (bind, acept...)

Nos dan un esqueleto donde están hechos socket, bind, listen y acept. Y la confguracion
de los structs raros. Utiza las funciones htons y tal que no estan permitidas en el subjetc.

Importante mirar el select. Como se configura el puerto deseado y tal. Y sobre todos
crear funcion de send y recv buenas.
